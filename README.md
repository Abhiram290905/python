What is Python?
* Python is a high-level, interpreted, and general-purpose programming language that is widely used because it is easy to learn, powerful, and very versatile.
* Python was created by Guido van Rossum and first released in 1991.
* It focuses on code readability and allows developers to write programs using fewer lines of code


# Key Features of Python

 1) Simple & Easy to Learn – English-like syntax
 2) Interpreted Language – No need to compile
 3) Dynamically Typed – No need to declare data types
 4) Object-Oriented – Supports classes and objects
 5) Cross-Platform – Runs on Windows, Linux, macOS
 6) Huge Standard Library – Saves development time
